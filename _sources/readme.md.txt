# Установка
1. Создайте новый проект на GitHub
- без README
- без .gitignore
Все эти файлы уже есть в шаблоне

Скопируйте ссылку на новый репозиторий (она пригодится вам на 5 шаге)

2. Клонируйте репозиторий на свой ПК

2. Запустите новое виртуальное окружение
python -m venv venv
3. Активируйте виртуальное окружение установите все пакеты
venv\Scripts\activate
pip install -r requirements.txt

4. Создайте файл .env в корне проекта и заполните его своими значениями в соответствии
с .env.example

4. Инициализируйте git в папке с проектом
git init

5. Удалите связи со старым репозиторием (шаблон) и добавьте связи с новым
git remote remove origin

git remote add origin https://github.com/maxlansky1/copier-test.git

6. Создайте ветку main и переключитесь на нее
git branch -M main

7. Влючите хуки pre-commit для проверки коммитов перед пушем

7. Создайте первый коммит и сделайте пуш на гитхаб. Не забывайте следовать руководству по коммитам
git add .
git commit -m "build: загрузка кода приложения на гитхаб"
git push -u origin main

8. После первого пуша зайдите в settings - pages - source - deploy from a branch

Выберите gh-pages /root, нажмите save. Теперь вся документация по вашему проекту будет доступна на Github Pages
